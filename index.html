What is a Loop?

Loops are fundamental constructs in programming languages that allow you to execute a block of code repeatedly. In Python, loops provide a powerful mechanism for iterating over sequences, performing repetitive tasks, and controlling the flow of your program. Here are some professional examples of Loops that you may see later down the line!

  • Processing Lists or Arrays
  • Iterating1 Through Files
  • Automating Repetitive Tasks

For Loops

A for loop statement that allows you to execute a block of code repeatedly, typically for a specified number of times or over a sequence of elements. It consists of three main components: initialization, condition, and iteration.

During initialization, you define a loop variable and set its initial value. The loop continues as long as the condition evaluates to true. The condition is typically a comparison involving the loop variable. With each iteration of the loop, the loop variable is updated according to the iteration expression. This expression can increment or decrement the loop variable, effectively controlling the loop's progress.

for i in range(5)
   print(i)

In this code, condition is "for each number in range(5)", so our loop will go around 5 times!

While Loops

A while loop statement allows you to execute a block of code repeatedly as long as a specified condition is true. It consists of a condition that is evaluated before each iteration, and if the condition evaluates to true, the code block is executed. The loop continues until the condition becomes false.

while condition:
   # code block to be executed

In this code, the loop will continue as long as the condition remains true. Once the condition becomes false, the loop will terminate.

Nested Loops

Nested loops are loops inside other loops. They are commonly used when you need to perform repeated actions within the context of another repeated action. For example, iterating over each item in a list and then performing another iteration within that loop.

for i in range(3):
   for j in range(2):
      # code block to be executed

In this code, the outer loop iterates 3 times, and for each iteration of the outer loop, the inner loop iterates 2 times. This results in a total of 6 iterations of the inner loop.

Loop Modifiers

Loop modifiers are special keywords or statements that allow you to control the flow of a loop and modify its behavior. They can be used to skip iterations, exit the loop prematurely, or perform specific actions based on certain conditions.

# Example of loop modifiers
for i in range(5):
    if i == 3:
        continue # Skip the current iteration
    if i == 4:
        break # Exit the loop
    print(i)

In this example:

  • The continue statement skips the current iteration of the loop when i equals 3.
  • The break statement exits the loop prematurely when i equals 4.
